Running Splits Average Time Calculator

This is a simple average calculator for calculating the average time for a set of running splits you've done.

To use it: enter your running times for each split, separated by a space. You can enter multiple sets of split times by entering each set of times followed by clicking 'Calculate'. The 'Reset' button is for erasing all previously entered times.

For example: If you've done a 800m x 6 workout just enter each of your split times separated by a space like so: '3:30 3:28 3:25 3:33 3:35 3:29'
